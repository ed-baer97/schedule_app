# Тестовые файлы

Эта папка содержит утилиты для тестирования и проверки системы.

## Файлы для перемещения

- `check_ai.py` - проверка подключения к AI API
- `quick_test.py` - быстрый тест миграции БД
- `test_imports.py` - проверка импортов перед миграцией
- `test_migration.py` - тест миграции

Эти файлы должны быть перемещены из корня проекта в эту папку.
